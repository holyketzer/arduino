# Parts

* Arduino Nano
* Power supply module (AC-DC 220V to 5V 600mA)
* Relay (like SRD-05VDC-SL-C)
* Analog temperature sensor (like KY-013)
* Rotary encoder (like KY-040)
* Any 2-digit semi-segment indicator x 2
* Extension cable
